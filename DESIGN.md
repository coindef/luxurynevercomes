# 买了个寂寞 · ParcelNeverComes — 产品设计文档

> 模拟购物，治愈消费冲动。不扣钱，不发货，只发多巴胺。

## 产品概述

「买了个寂寞」(ParcelNeverComes) 是一个像素级认真的假电商 SPA：完整复刻中文电商的全部肌肉记忆——瀑布流、秒杀倒计时、满减凑单、支付密码、物流推送——但把每个"花钱节点"反转成"省钱节点"。用户尽情浏览、加购、庄严支付 ¥0.00，多巴胺在期待侧全额释放；支付瞬间划线价翻转成绿色数字存入"省钱账本"，完成"冲动→仪式→安抚→沉淀"的治愈闭环。留存靠三个钩子：物流剧场（包裹永远在一条越来越哲学的路上，每天更新一集）、省钱账本（克制的可视化复利）、N 天后回访时刻（"钱还在你那儿，冲动已经不在了"）。调性 deadpan：嘲笑的永远是电商套路本身，落点永远是"你没花钱，你赢了"。纯前端 React + localStorage，移动优先，6 页。

## 页面结构

### 商城首页·拼命买 `/`

复刻淘宝/拼多多首页的信息轰炸密度，3 秒进入'逛'的心流；常驻省钱徽章提供正反馈入口

- 开屏新人红包弹窗：¥888 无门槛券，撕开有微动效（仅首次访问）
- 顶部搜索栏 placeholder 轮播：'搜什么都不会发货' / '大家都在搜：寂寞、空虚、一夜暴富'
- 限时秒杀横滑区：倒计时永远停在 00:59:XX（每次刷新重置），商品卡带'已抢87%'进度条
- 分类金刚区（emoji 图标）：数码家电 / 家居日用 / 穿搭配饰 / 美妆个护 / 零食饮品 / 玄学好物
- 双列瀑布流商品卡：emoji+渐变色块封面、划线原价、'最后3件'红角标、'包邮'标签
- 滚动跑马灯：'北京用户 2秒前 下单了 1 件寂寞'
- 常驻省钱徽章：'已为你守住 ¥X'，点击直达省钱账本
- 底部 Tab 栏：首页 / 购物车（跳动角标）/ 订单 / 我的

### 商品详情页·上头 `/product/:id`

用全套详情页仪式（价格爆炸、库存压迫、好评如潮）把'想要'推到顶点，冲动需要被认真对待

- 大图区：emoji 在渐变色块上缓慢浮动，轮播小圆点（每张图一模一样，彩蛋）
- 价格区：秒杀价 + 划线原价 + '比你收藏时又降了 ¥X'
- 双压迫条：秒杀倒计时 + 库存条'仅剩3件'（小字：三年了，一直剩3件）
- 服务保障栏：'七天无理由不发货 · 假一赔十（都是假的）· 全场包邮（邮不出去）'
- 实时下单弹幕从底部飘过：'广州用户 3秒前 拍下 1 件'
- 评价区'好评如潮(10万+)'：预置 deadpan 好评池
- 吸底操作栏：收藏 / 加入购物车（黄）/ 立即拿下（大红，呼吸动效）

### 购物车·凑单的艺术 `/cart`

把满减凑单从被动套路变成主动玩法——真电商里凑单让你多花钱，这里只产生纯粹的进度条快感

- 商品清单：可改数量、可删除、可勾选
- 满减进度条：'还差 ¥38 享满199减50'，随勾选实时增长
- 进度条凑满瞬间：小烟花动效 + toast '凑单成功，这波血赚'
- 底部结算栏：合计价 + 醒目绿字'本单为你守住 ¥X'
- 空购物车状态：插画 + 文案钩子引导回首页

### 结算与支付成功·¥0.00 的仪式 `/checkout`

全站多巴胺峰值。完整收银台仪式 + 支付成功的金额翻转入账，是产品论点的具象化：仪式是真的，代价是零

- 收货地址：预设'我家 / 我心里 / 虚空3号楼'，默认'就送到心里吧'
- 配送方式：寂寞速运（永不达）¥0 / 虚无次日达（次日也不达）+¥0
- 冲动来源小问句 + 可选 chips：加班了 / 被夸了 / 没被夸 / 天冷 / 发工资了 / 就是想买（可跳过），选中只回一句'记下了。'
- 优惠明细逐行划减动画：原价 → 秒杀减免 → 券 → 实付 ¥0.00
- 大红按钮'支付 ¥0.00' + 假指纹/密码动画（随便按都对）
- 支付成功页（同路由切状态）：绿色对勾 + emoji 彩带（🕳️🕊️💸），先弹'-¥0.00'，停顿半秒，原价翻转成绿色'+¥X 已存入省钱账本'滚动累加
- 小票样式卡片：订单号 JM-2026-空空如也-XXXX，'预计送达：等一个契机'，按 9:16 截图安全区构图
- 轮换安抚语卡片（文案池，按冲动标签轻度匹配）+ 双出口：'查看物流（它不会来）' / '再买点寂寞'

### 订单与物流剧场·永远在路上 `/orders`

核心留存引擎：等快递的期待感被做成永不落地的连载，节点按真实时间解锁，荒诞浓度递增，第7天收束成治愈

- 订单列表：状态永远'运输中'带小货车 icon，第7天后显示'已抵达心里'；卡片上有'这单已陪你 N 天'
- 订单详情（同页展开）：物流时间轴按真实时间解锁，文案从物流腔逐日滑向哲学腔（9 级剧本）
- 骑手卡片：'骑手·加缪'，状态'正在思考配送的意义'，拨打电话显示'对方也很寂寞，未接听'
- '催一催'按钮：随机 deadpan 回复池，可无限催
- '确认收货'按钮永远置灰，长按弹出'收不到的，这就是本店的服务'
- 回访安抚语：打开旧订单顶部浮现（3天/7天/30天各一套）
- 空订单状态 + 页脚：'你的寂寞只存在这台设备上，我们看不到，也寄不出'

### 我的·省钱账本 `/me`

情绪闭环页 + 长期正反馈中枢：把'什么都没买到'郑重翻译成'你守住了钱'，全站唯一收起玩笑的真诚时刻

- 头像 + 会员等级：'寂寞黑卡会员'（按下单次数升级）
- 累计守住总额超大数字，缓动滚动动画
- 折算参照物轮换：'≈ 32 杯奶茶' / '≈ 1.5 张去大理的机票'，可点击换算
- 冲动来源回顾：'你最常在周四深夜想买东西'式温和洞察，只陈述不建议
- 订单快捷入口 + 本月冲动次数统计：'本月你冲动了 23 次，守住了 ¥4,281'
- 设置：清空所有寂寞（清 localStorage，二次确认文案格外温柔）

## 招牌时刻

**¥0.00 支付仪式与金额翻转** — 全站多巴胺峰值，三派共识合成版。完整复刻收银台肌肉记忆：假指纹动画、'叮'的到账音效、绿色对勾、emoji 彩带——先弹'-¥0.00'，停顿半秒，划线价翻转成绿色'+¥299 已存入省钱账本'滚动累加。扣款这个最心痛的瞬间被原样保留了仪式，却反转成了获得。整页按 9:16 截图安全区构图，是转发率最高的一屏。

**物流追踪剧场·骑手加缪** — 核心留存引擎。包裹按真实时间推进，节点从'已揽收，重量 0g'的正经物流腔逐日滑向'骑手遭遇哲学问题：既然一切都会消失，为什么要配送'，第 7 天终章'包裹已送达你心里，无需签收'把梗收束成真诚治愈——落空被写成了抵达。用户会为了看下一集而回访，回访即复用，复用即再次治愈。

**永远置灰的确认收货** — 全站唯一不能点的按钮，产品哲学的实体化。长按弹出'收不到的，这就是本店的服务'。用户第一次发现它不能点的瞬间，是从'逛假淘宝'到'读懂这个网站'的顿悟时刻，天然适合录屏传播。

**无限催单扭蛋机** — 催单按钮永远有效，骑手永远有新回复（'我在很努力地不来了' / '催也是一种陪伴'），把网购最焦虑的动作变成扭蛋机。实现成本极低（一个随机文案池），产出稳定的社交货币。

**薛定谔的秒杀 + 永远剩3件** — 秒杀倒计时永远停在 00:59:XX，每次刷新重置；所有商品永远'仅剩3件'，详情页小字注明'三年了，一直剩3件'。稀缺话术照常起效（拟真），戳破方式让用户和产品站在同一边嘲笑套路本身（冷幽默），最终变成安心：这里的'限时'永远不会真的结束（治愈）。一个机制吃三层。

**满减凑单烟花** — 凑单是中文电商最独特的上头行为。满减进度条随勾选实时增长，凑满瞬间放小烟花 + toast'这波血赚'。真实电商里凑单让你多花钱，这里凑单只产生纯粹的进度条快感，是'套路变糖'的最佳样本。

**冲动来源的温柔一问** — 结算页一行小字'今晚想买，是因为——'，几个 chips 轻轻摆着，可跳过。选了也只回一句'记下了。'，不分析不建议。标签留在订单里，一个月后回看就是一本没打算写的情绪日记。被理解的前提是被记住，而不是被纠正。

**N 天后的回访时刻** — 打开旧订单，顶部浮现'这单已陪你 14 天。钱还在你那儿，冲动已经不在了。'全产品唯一接近'点破'的时刻，由时间说出口而非产品说教，所以成立。它让用户亲眼确认：当晚那个非买不可的东西，不买也真的没事。

## 视觉方向

> 2026-07-13 更新：商品图已从「emoji + 渐变色块」升级为真实商品照片（本地 `public/img/`，加载失败时回退 emoji 色块）。真实照片让"一本正经的假电商"更可信，荒诞商品（量子快递盒、一整个春天）配实拍图反而更好笑。

主视觉走"电商大红橙系"——这不是偷懒，是产品论点的地基：界面必须一眼像淘宝/拼多多，用户的电商肌肉记忆才能被完整激活，deadpan 文案的荒诞才有落差可言，走清新独立路线反而会杀死笑点。具体：主色用电商红 #FF3B30 → 橙 #FF6A00 渐变（秒杀区、下单按钮、角标、倒计时），辅以促销黄 #FFD54D（加购按钮、优惠券）；但保留一条"治愈色轨"——所有省钱/安抚相关元素（省钱徽章、+¥X 入账数字、账本页、安抚语卡片）统一用治愈绿 #22A06B + 暖米白 #FFF9F0，形成"红色是仪式、绿色是真心"的双色语义系统，用户会无意识学会这个规则。商品图 = 大号 emoji（56-72px）居中于低饱和双色渐变色块（每个分类一组固定渐变，如数码=蓝紫、玄学=暗夜蓝金），圆角 12px。字体用系统中文字号栈（PingFang SC / HarmonyOS Sans），价格与账本数字用 tabular-nums 等宽数字加粗。布局：375px 移动容器，桌面端居中 + 柔和投影卡片壳 + 页面外留白，容器内信息密度拉满（拟真），容器外干净克制（本店人设）。动效原则：促销侧动效浮夸（呼吸按钮、烟花、撒花），治愈侧动效克制（数字缓动滚动、卡片淡入），一浮夸一克制正是产品的双重人格。

## MVP 范围

- 1. 项目脚手架：Vite + React + React Router，375px 移动容器布局壳（桌面居中），设计 token（红橙促销色 + 治愈绿双色系统）
- 2. 数据层：30 个 mock 商品数据文件（emoji + 渐变色块映射 + 价格 + 划线价）；localStorage store（购物车、订单、累计省钱额、冲动标签），含订单时间戳供物流剧本推进
- 3. 商城首页：分类金刚区、秒杀横滑区（永远 00:59:XX 倒计时）、双列瀑布流商品卡、跑马灯、新人红包弹窗（首访一次）、底部 Tab 栏、常驻省钱徽章
- 4. 商品详情页：渐变大图区、价格爆炸区、'仅剩3件'压迫条、服务保障栏、预置好评区、下单弹幕、吸底操作栏（加购/立即拿下）
- 5. 购物车页：清单增删改、满减进度条 + 凑满烟花 toast、结算栏'本单为你守住 ¥X'、空状态
- 6. 结算 + 支付成功（同路由双状态）：地址预设、配送方式、冲动来源 chips、明细划减动画、¥0.00 支付按钮 + 假指纹动画、成功页对勾 + emoji 彩带 + 金额翻转入账动效 + 小票卡片 + 安抚语轮换
- 7. 订单与物流剧场：订单列表（'这单已陪你 N 天'）、9 级物流剧本按真实时间解锁、骑手·加缪卡片、无限催单随机回复池、永远置灰的确认收货（长按彩蛋）、旧订单回访文案（3/7/30 天）
